# Entrega Semanas 1-4

## Semanas 1-2

- configuracao do monorepo backend
- definicao do modelo inicial de usuarios e pacientes
- configuracao de Docker Compose
- provisionamento local de PostgreSQL e RabbitMQ
- documentacao de arquitetura e execucao

## Semanas 3-4

- servico de usuarios e autenticacao com JWT
- servico de pacientes com API REST
- API Gateway para entrada unificada
- notificacoes assincronas basicas via RabbitMQ
